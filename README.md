# 🌦️ Weather Report Application

A responsive **Weather Report Application** built with **React.js**, **Tailwind CSS**, and **Axios** that allows users to search for real-time weather information for any city. The application fetches live weather data from a Weather API and displays essential details such as temperature, weather conditions, humidity, wind speed, and more in a clean, user-friendly interface.

The project demonstrates how to integrate third-party APIs into a React application, manage state using React Hooks, handle asynchronous API requests with Axios, and implement responsive UI design using Tailwind CSS. It also includes basic error handling to provide a smooth user experience when invalid city names are entered or API requests fail.

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Display real-time weather information
* ☁️ Show temperature, weather condition, humidity, and wind speed
* ⚡ Fetch live data using Axios and Weather API
* 🎨 Modern and responsive UI built with Tailwind CSS
* 📱 Mobile-friendly design using Flexbox/Grid and responsive layouts
* ⚛️ Built with React functional components and Hooks (`useState`)
* ❌ Basic error handling for invalid city names and API failures
* 🧩 Organized component-based architecture for better maintainability

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6+)
* Tailwind CSS
* Axios
* Weather API (OpenWeatherMap / WeatherAPI)
